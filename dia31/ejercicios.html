Eventos en JavaScript

Una lista de eventos comunes y útiles en JavaScript

Lista de eventos

click
Se activa cuando un usuario hace clic en un elemento de la página.
mouseover
Se activa cuando el puntero del mouse pasa sobre un elemento.
mouseout
Se activa cuando el puntero del mouse deja de estar sobre un elemento.
keydown
Se activa cuando el usuario presiona una tecla en el teclado.
keyup
Se activa cuando el usuario suelta una tecla en el teclado.
keypress
Se activa cuando el usuario presiona una tecla, pero solo para teclas con caracteres imprimibles.
focus
Se activa cuando un elemento recibe el foco, como un campo de entrada o un área de texto.
blur
Se activa cuando un elemento pierde el foco, como un campo de entrada.
submit
Se activa cuando se envía un formulario (por ejemplo, al hacer clic en un botón de enviar).
change
Se activa cuando el valor de un elemento de formulario cambia, como un campo de texto o un menú desplegable.
input
Se activa cada vez que el usuario introduce texto en un campo de entrada o realiza un cambio en un elemento editable.
load
Se activa cuando un recurso, como una imagen o un archivo, se ha cargado completamente.
resize
Se activa cuando la ventana del navegador o un elemento cambia de tamaño.
scroll
Se activa cuando el usuario desplaza la página o un contenedor de desplazamiento.
drag
Se activa cuando un elemento es arrastrado dentro de la ventana del navegador.
drop
Se activa cuando un elemento arrastrado se suelta sobre un área de destino.
touchstart
Se activa cuando un usuario toca la pantalla de un dispositivo móvil o táctil.
touchend
Se activa cuando un usuario levanta el dedo de la pantalla de un dispositivos móvil o táctil.
